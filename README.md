# Back-end

### Description

```
2차 프로젝트 7팀 백엔드 레포지토리
```

### 서버 실행 방법

```bash
git clone

cd back-end

npm install

npm run dev
```

### Commit 컨벤션

```
Feat: 새로운 기능 추가
Fix: 코드 및 버그 수정
Style: 코드 포맷 변경 등 기존 코드에 영향이 없는 경우
Comment: 주석 추가 및 수정
Docs: README.md 등 문서 수정
Rename: 파일명 수정
Remove: 파일 삭제
```
